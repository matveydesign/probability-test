How to use :

First select a Tab on the left, The General tab is loaded with sample data
Load the network with previous winnings to generate a model in the Large Text area
1. First, click on [Create Groups] button.
2. Then, Select a group you want to use
3. Then, click on [Create Neural Network] button.
4. Then, click [Read Training Data].
5. Then, click [Run] button as many times as you like to see trained network and results of run.



Additional options

Reference date - If selected it will take additional column form the winnings to identify where the winning is in the list, if not selected the computer will set this automatically.

Number of columns in a row - This input tells the app how many numbers make a row of winning numbers.

Number on balls go to - How High do the numbers on the balls count up to.

Configure your groups (10,20,30,40,50) - Grouping up winning numbers into sets, this input takes a rule for how you want the sets to be created, for example if you insert '10,50' and a winning ticket is ' 3 19 50 49 24' then the group will be '1-4' because there is only 1 number less than or equal to 10 and the remain 4 numbers are greater than or equal to 50

Hidden Layers first learning Shape - Brain.js neural network model configuration, by default you can set this to 70, if you want to adjust the apps predictions you can insert more complex models ex '5,5,2,9'.

Predict next group - if true, this will perform a sequence prediction on the groups based on each winning row and when the group was created to predict the next posible group.
